# Database Dashboard
CRUD operations, routines execution (functions, procedures), console | PostgreSQL

With logging feature

### Run
> cd src

> docker compose build

> docker compose up

## Languages & Tools
- ASP.NET Core MVC
- PostgreSQL
- Docker + Docker compose
- Nginx
- Bootstrap
